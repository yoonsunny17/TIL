# CRUD 정리하기

|       | 구문   | 예시                                                         |
| ----- | ------ | ------------------------------------------------------------ |
| **C** | INSERT | `INSERT INTO <tablename> (col1, col2, ...) VALUES (val1, val2, ...);` |
| **R** | SELECT | `SELECT * FROM <tablename> WHERE 조건;`                      |
| **U** | UPDATE | `UPDATE <tablename> SET col1=val1, col2=val2, ... WHERE 조건;` |
| **D** | DELETE | `DELETE FROM <tablename> WHERE 조건;`                        |
